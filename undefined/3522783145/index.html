about proxy bypass

通过大量抓取可用代理

使用clash作为代理软件，设置config.yaml,以下为攻击部分配置，可参考配置

日常使用时，使用一个配置，在需要对目标进行测试时，使用另外的配置

关于如何抓取代理

  • github
  • 通过fofa，shodan之类的搜索引擎，搜索当前区域内的1080等常见端口，进行抓取代理
  • 抓取肉鸡，配置代理，不建议，因为肉鸡与当前网络连通性是否良好不确定

    抓取代理后，进行验证

    将抓取结果转成 json格式，[{“protocol”:”socks5”,”ip”:”5.189.153.156”,”prot”:1080},]
    验证方式：curl -x socks5://ip:port www.google.com
    如抓取时，未对地区进行校验，建议使用geoip2的ip数据库，对ip地区进行检查，再进行代理验证
    使用json格式(为啥使用json格式，是由于github上公开的代理返回格式为json格式，简单来说就是懒)，然后使用python3进行批量验证，之后会写一个完善的脚本就想代理获取检查

    代理获取

  • github代理提取
